# Changelog

## Version 1.0.0
### Features
- **Home Page**
  - Hero section with split layout and dual CTAs.
  - Trust bar displaying accepted insurances.
  - Interactive Pain Map feature.
  - Services preview with service cards.
  - Testimonials section with masonry layout.
  - Team preview with asymmetrical layout.
  - Call-to-action section with a mini-form.

- **About Page**
  - Hero section with a full-width image overlay.
  - Philosophy section with pull quotes.
  - Technology grid showcasing dental technology.
  - Team section with arch-shaped image masks.

- **Services Page**
  - Hero section with a search bar.
  - Accordion style service list.
  - Zig-zag layout for featured services with finance calculator.

### Known Limitations
- Currently, the testimonials section does not support dynamic loading of new testimonials.
- The finance calculator uses a static rate and does not integrate with real-time data.
